# Yvonne Hospitality & Management

Personal hospitality portfolio website for **Yvonne Agane Mugasia**.

## Tech Stack

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: CSS Modules + CSS custom properties
- **Animations**: Framer Motion
- **Icons**: React Icons
- **Routing**: React Router v6

## Local Development

```bash
cd frontend
npm install
npm run dev
```

Visit `http://localhost:5173`

## Build

```bash
cd frontend
npm run build
```

Output goes to `frontend/dist/`

## Deployment

This project is configured for **Render** (static site).  
See `render.yaml` at the root for build settings.
